STL之Lists1

List 是存储在Linked list中的序列元素，与Vectors相比它能快速插入和删除元素，但是随机访问比较慢。

List 构造函数

默认构造函数没有参数，创建一个新的实例化对象。

第二个是默认复制函数，创建一个新的实例对象，并初始为c中元素。

第三个是创建一个带有num个对象空间的list，如果val指定值，则初始化值为val。

任何一个list都有 默认构造、赋值构造以及析构函数。

List 运算符重载

执行比较和赋值都需要线性的时间

List 函数

assign

将list 赋值为num个val; 或者list赋值为从start到end的元素

此函数将破坏list的先前内容。

back

返回list中最后一个元素的引用

begin

返回一个迭代器指向list的首个元素，常数时间。

clear

删除list所有元素，常数时间。

empty

判断list是否为空。

end

返回一个迭代器指向list的追后一个元素的下一个位置。

eras

擦除一个元素或[start,end)个元素。返回擦除的最后一个元素。

front

返回第一元素。

insert

  • 插入val在loc之后，放回插入元素位置的迭代器。
  • 插入num个val 在loc之后。
  • 插入[start,end) 区间内的元素，在loc之后。

max_size

返回list在更多内存的情况下，允许分配的最大值。

需要与size()和capacity()区分

merge

合并list与lst 默认以<的方式合并，如果指定compfunction，则遵循compfunction的对比方式。

pop_back

删除最后一个元素。

pop_front

删除第一元素

Vector 没有该函数

push_back

添加一个元素到list尾部。

push_front

Vector 没有该函数

remove

删除所有等于val的元素。

Vector 没有该函数

remove_if

删除所有pr为true的元素。

Vector 没有该函数

rbegin

返回一个迭代器指向list末尾的下一个元素。

rend

返回一个迭代器指向list第一个元素。

reserve

设置list的容量最小大为size。

size

返回list中元素的个数。

sort

以升序的方式排序list，如果指定p这按照该方式排序。

Vector 没有该函数

splice

在pos的位置插入lst，或者在插入list中的del位置的元素。抑或插入start到end中的元素。

在参数时间级别完成

Vector 没有该函数

swap

将list的内容与from的内容进行交换，常数时间。

unique

移除list中所有连续重复的元素，去重需要先排序。

使用==进行比较，除非指定pr。

调用后原有元素顺序不会改变。

Vector 没有该函数

Lists的就先到这里吧，Lists和Vector有很多函数都是相似的可以对比着学习
